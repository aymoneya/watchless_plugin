------THIS IS THE README FILE FOR MY LAB3 PLUGIN------------

plugin: Watchless-Plugin


Widget: Weekly Archives 
Displays a weekly archive of posts.



shortcode 1: 

Night Mode
Inserts an ON and OFF buttons to a post, 
when on is clicked: the websites background becomes white, the text becomes black.

when off is clicked: the websites background becomes black, the text becomes white.

This short code creates a "night mode" where users are able to view the website more comfortably at night/with little light
It is also useful for saving phone battery, as less brightness is used.

usage: [night_mode][/night_mode]





shortcode 2: 

google Maps

inserts a frame of google maps, given width/height/source 

link
usage: [googlemaps width="" height="" source=""]
